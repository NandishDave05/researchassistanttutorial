Data Analysis

Looking for Patterns and Relationships in the Data

Source: Cienpies Design/Shutterstock.com

Many researchers set out to show how two variables are related, for example, smoking and cancer, fitness and heart health, or studying and marks. In this next section, you will get a general sense of some of the tests that might be used to show relationship between data sets.

Let’s say some researchers would like to know if tutoring has an impact on students’ marks. They hypothesize that for every hour spent with a tutor, a student’s marks will improve.

In order to test this theory, they will need to find some way to show that the two values (hours spent with tutor + change in marks) are related in some significant way; that as hours go up, so do marks. Easier said than done!

For example, there could be other confounding variables that are contributing to the change in both hours with tutor and marks. What if the students must pass a healthy restaurant on their way to visit the tutor and often buy a nourishing sandwich? Maybe the dietary change – the physical exercise during the walk – is responsible for any improvement in their marks. Or, maybe any simultaneous increase in both values is the result of pure fluke.

Often, researchers want to know if their sets of data show significant association. This term suggests that as one value changes the other changes as well. Significant association is strengthened when large sample sizes are used, but statistical tests can be performed on data collected from small groups too. These tests, sometimes called chi-tests, t-tests, or the Mann-Whitney U test, can be applied to multiple sets of data to show if they are related beyond what could occur by chance alone.

Showing that one thing causes another is very tricky indeed. If researchers want to show that one variable is dependent on another (in other words, that one variable causes changes in another), they employ what is known as a regression analysis. A regression analysis will show if value X changes when value B changes and which value is responsible for triggering the change.

In your role as a research assistant, ask your research supervisor which technique he or she plans to use to analyze the data. The more you understand about the particular type of data analysis you will be using, the greater your contribution will be to the project.
